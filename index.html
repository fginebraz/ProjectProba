Pregunta 4

Se lanzan 3 dados balanceados. Sea X la suma de las caras. (a) Para n = 100, 1000, 10000, 100000 haga n simulaciones para obtener la función de masa de probabilidad aproximada de X. (b) Usando las funciones de las librería ‘patchwork’ y ‘ggplot2’ ponga en un mismo gráafico las cuatro gráficas de las funciones de masa del inicio anterior c) Para n = 10, . . . , 100000 haga n simulaciones para obtener un estimado de la probabilidad P(X ≤ 3). Ponga en un gráfico n en el eje horizontal y la probabilidad en el eje vertical.

Definimos las caras del dado y el número de dados del problema:

caras_del_dado <- 1:6
num_dados <- 3

Definimos una función que nos haga las simulaciones que toma como input el número de iteraciones que queremos que realice:

simuleichon <- function(num_sim){
  vector_simuls <- c()
  for(i in 1:num_sim){
    simulacion <- sample(x = caras_del_dado, size = num_dados, replace = TRUE)
    valor_simulacion <- sum(simulacion)
    vector_simuls <- c(vector_simuls,valor_simulacion)
  }
  print(paste("Los primeros valores (no tiene sentido imprimirlos todos) de la función de masa de probabilidad son", str(vector_simuls)))
  df <- data.frame(suma = vector_simuls)
  head(df)
  grafica1 <- df %>% ggplot() +
    geom_histogram(aes(x = suma,
                       y=..count../sum(after_stat(count))),
                   linewidth=1, fill = "cadetblue2") +
    ggtitle("Densidad del producto n=", num_sim) +
    ylab("f") + xlab("suma") + theme_light()
  
  grafica1

  return(list(grafica1,vector_simuls))
}

probabilidades_estimadas <- c()
for(i in 1:length(simuls_satisfacen)){
  satisfacen <- simuls_satisfacen[1:i]
  proba_estimada <- sum(satisfacen)/length(satisfacen)
  probabilidades_estimadas <- c(probabilidades_estimadas, proba_estimada)
}
